ABOUT — 关于我

一个不安分的
工业工程师

「产线教我看见问题，
代码给我解决问题的自由。」

我是李汶翰，Aircom 宁波的 IE 工程师，负责 CA / CP / Hose 三条汽车零部件产线的工艺与改善。白天在现场调焊接参数、导自动化线、迎国际审核；工位之外，我把这些真实痛点一个个写成软件——故障判定、视觉检测、审核系统、PLC 监控——让工厂里的经验变成可以复用的工具。

供职于欧企，英语为工作语言，直接对接 Volvo、Audi、Subaru、BMW、Nissan 等主机厂需求端。

  1. 焊接工艺深耕超声波 / 电阻焊接工艺，在降噪、提速、外观与强度优化上有大量一线经验
  2. 自动化导入 PM主导自动化产线替代手工产线全过程，把控项目节点与深度风险管控
  3. 视觉防错从零导入KEYENCE / BALLUFF 视觉系统选型、调试到量产，识别判定经验丰富
  4. 体系审核应审深度参与 IATF 16949 / ISO 9001 / 14001 / 45001 审核，保持 100% 通过
  5. 低成本改造用最小投入完成产能、OEE、PPPH 的提升与人机工程优化
  6. 软件自研能力Python / Web 全栈 / 机器视觉 / PLC 通讯，把现场问题写成产品级工具
